Node环境管理

1.用homebrew安装nvm

brew install nvm

2.创建Node的安装目录

mkdir ~/.nvm

3.在~/.bash_profile文件中添加以下代码

export NVM_DIR=~/.nvm
. $(brew --prefix nvm)/nvm.sh
4.可以用nvm安装node
nvm install node
